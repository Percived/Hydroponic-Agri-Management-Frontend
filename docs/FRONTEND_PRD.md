# 水培农业管理系统 Web 前端 PRD

## 1. 项目概述

### 1.1 项目背景
水培农业管理系统 Web 前端是为温室/大棚水培环境设计的管理平台，支持设备管理、实时监控、数据采集展示、控制指令下发等功能。系统需支持 PC 端与移动端访问。

### 1.2 技术栈
- **框架**: Vue 3 + Composition API
- **UI 组件库**: Element Plus
- **构建工具**: Vite
- **状态管理**: Pinia
- **路由**: Vue Router
- **HTTP 客户端**: Axios
- **图表**: ECharts
- **语言**: TypeScript（可选）

### 1.3 目标用户
| 角色 | 权限范围 |
|------|---------|
| 管理员 (ADMIN) | 全部功能：用户管理、设备管理、控制操作、告警处理、系统配置 |
| 操作员 (OPERATOR) | 查询功能 + 控制操作 + 告警处理 |
| 只读用户 (VIEWER) | 仅查询功能 |

## 3. 功能模块详细设计

### 3.1 登录模块 (P0)

#### 3.1.1 登录页
**页面路径**: `/login`

**功能描述**:
- 用户名/密码登录
- 登录成功后存储 JWT Token 并跳转首页
- 登录失败显示错误提示

**交互规则**:
1. 用户名：3-32 字符，必填
2. 密码：6-64 字符，必填
3. 登录按钮点击后显示 loading 状态
4. 登录失败：显示错误信息（用户名或密码错误、账户已禁用等）
5. 登录成功：存储 Token 到 localStorage，跳转首页

**API 接口**: `POST /api/auth/login`

**响应处理**:
```json
{
  "code": 0,
  "data": {
    "token": "jwt.token.here",
    "expires_in": 7200,
    "user": {
      "id": 1,
      "username": "admin",
      "roles": ["ADMIN"]
    }
  }
}
```

**移动端适配**:
- 居中卡片布局
- 响应式宽度调整

### 3.2 设备管理模块 (P0)

#### 3.2.1 设备列表页
**页面路径**: `/devices`

**功能描述**:
- 展示所有设备列表，支持分页
- 支持按类型、分类、分组、状态筛选
- 支持关键词搜索
- 管理员可新增、编辑、启用/禁用设备

**页面布局**:
```
┌─────────────────────────────────────────────────────────────┐
│ 设备管理                                    [新增设备] 按钮   │
├─────────────────────────────────────────────────────────────┤
│ 筛选区:                                                      │
│ [类型 ▼] [分类 ▼] [分组 ▼] [状态 ▼] [搜索关键词] [查询] [重置] │
├─────────────────────────────────────────────────────────────┤
│ 数据表格:                                                     │
│ ┌────┬──────────┬──────┬──────┬──────┬──────┬───────┬──────┐ │
│ │ ID │ 设备编码  │ 名称  │ 类型  │ 分类  │ 分组  │ 状态  │ 操作 │ │
│ ├────┼──────────┼──────┼──────┼──────┼──────┼───────┼──────┤ │
│ │101 │ DEV-001  │ 温度1 │ 传感器│ TEMP │ A区   │ 🟢在线 │ 详情 │ │
│ │102 │ DEV-002  │ 风机1 │ 执行器│ FAN  │ A区   │ 🔴离线 │ 详情 │ │
│ └────┴──────────┴──────┴──────┴──────┴──────┴───────┴──────┘ │
│                                                              │
│ 分页: [<] 1 2 3 ... [>]  共 50 条                            │
└─────────────────────────────────────────────────────────────┘
```

**字段说明**:
| 字段 | 说明 | 示例 |
|------|------|------|
| ID | 设备 ID | 101 |
| 设备编码 | 唯一编码 | DEV-001 |
| 名称 | 设备名称 | 温度传感器1 |
| 类型 | SENSOR/ACTUATOR | 传感器/执行器 |
| 分类 | 设备分类 | TEMP, HUMIDITY, FAN 等 |
| 分组 | 所属分组 | A区 |
| 状态 | 在线/离线/启用/禁用 | 🟢在线/启用 |
| 操作 | 详情/编辑/禁用 | - |

**筛选条件**:
| 字段 | 类型 | 选项 |
|------|------|------|
| 类型 | 下拉 | 全部/传感器/执行器 |
| 分类 | 下拉 | 全部/TEMP/HUMIDITY/... |
| 分组 | 下拉 | 全部/各分组名称 |
| 状态 | 下拉 | 全部/启用/禁用 |
| 关键词 | 输入框 | 搜索设备编码或名称 |

**API 接口**: `GET /api/devices`

**权限控制**:
- VIEWER: 仅查看
- OPERATOR: 查看详情
- ADMIN: 新增、编辑、启用/禁用

**移动端适配**:
- 表格改为卡片列表
- 筛选区折叠为抽屉
- 操作按钮固定底部

| 设备编码 | Input | 是 | 1-64字符，唯一 |

**API 接口**: `POST /api/devices` (新增) / `PUT /api/devices/{deviceId}` (编辑)
